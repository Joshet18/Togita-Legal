Política de Privacidad

Esta Política de Privacidad describe cómo se recopila, utiliza y protege la información de los usuarios del bot Togita Himiko.

1. Información que recopilamos

El bot puede recopilar la siguiente información cuando es utilizado:

  • ID de usuario de Discord
  • ID de servidor (guild)
  • Datos necesarios para el funcionamiento del bot (por ejemplo, configuraciones personalizadas)

No recopilamos información personal sensible ni almacenamos mensajes privados.

2. Uso de la información

La información se utiliza únicamente para:

  • Proveer las funcionalidades del bot
  • Mejorar el servicio
  • Identificar problemas y errores

3. Compartición de datos

No compartimos, vendemos ni alquilamos información de los usuarios con terceros, salvo obligación legal.

4. Almacenamiento de datos

Los datos se almacenan de forma segura y se conservan solo el tiempo necesario para el funcionamiento del bot.

5. Derechos de los usuarios

Si deseas que eliminemos tus datos, puedes contactarnos en: togita.himiko18@gmail.com

6. Cambios a esta política

Nos reservamos el derecho de actualizar esta política. Los cambios serán notificados a través del servidor de soporte o el bot.
